    Projetos em Ciência de Dados

    Identificação de imóveis revenda para maximizar o lucro

    O Projeto House Rocket é um projeto do tipo Insights fictício que busca analisar as melhores oportunidades de compras de imóveis para a empresa House Rocket (também fictícia). Utilizando Python e Estatística estudei os dados afim de criar algumas hipóteses de negócios e recomendar os 882 melhores imóveis para compra. Com uma margem de lucro de 30% estima-se um lucro de U$108 milhões.

    • Python, Pandas, Numpy e Seaborn.
    • Anaconda, Pycharm e Jupyter Notebook.
    • Mapas interativos com Plotly e Folium.
    • Heroku Cloud.

    Previsão de vendas das Lojas Rossmann

    O Projeto das Lojas Rossmann é um projeto de Ciências de Dados para a predição de vendas da gigante farmaceutica européia Rossmann. O objetivo do projeto é prever as vendas de cada loja Rossmann nas próximas seis semanas. Utilizando o modelo de regressão XGBoost, fizemos a previsão com erro de aproximadamente 10%. O modelo está publicado em produção e pode ser acessado em qualquer horário por um Bot do Telegram.

    As ferramentas utilizadas foram:

    • Git e Github.
    • Python, Pandas, Matplotlib, Seaborn.
    • Jupyter Notebook.
    • Regressões:Linear, Lasso, Random Forest e XGBoost.
    • BOT no Telegram.
    • Cloud Heroku.
