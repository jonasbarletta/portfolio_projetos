Projetos em Ciência de Dados

Identificação de imóveis para compra e revenda a fim de maximizar o lucro

O Projeto House Rocket é um projeto do tipo Insights fictício que busca analisar as melhores oportunidades de compras de imóveis para a empresa House Rocket (também fictícia). Utilizando Python e Estatística estudei os dados afim de criar algumas hipóteses de negócios e recomendar os 882 melhores imóveis para compra. Com uma margem de lucro de 30% estima-se um lucro de U$108 milhões.

As ferramentas utilizadas foram:

  • Python, Pandas, Numpy e Seaborn.
  • Anaconda, Pycharm e Jupyter Notebook.
  • Mapas interativos com Plotly e Folium.
  • Heroku Cloud.
  • Streamlit Python framework web.

Catalogar opções de calça jeans masculinas via web scraping

Com o objetivo de catalogar as opções de calça jeans do mercado utilizei o pacote Beautiful Soup do Python para fazer um web scraping do site da H&M. Com a ferramenta de orquestração de tarefas Cron e o Banco de Dados SQLite, coletamos os dados automaticamente por uma semana. Após esse período analisamos os dados afim de entender opções de calça jeans masculinas.

As ferramentas utilizadas foram:

  • Python, Pandas e Beautiful Soup
  • Jupyter Notebook, Pycharm
