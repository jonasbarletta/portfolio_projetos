    Classificação de Clientes da Seguradora Insurance All.

    A empresa (fictícia) Insurance All oferecerá aos seus clientes de seguro de vida um seguro de automóvel no modelo de venda cruzada. Com a base de dados da empresa construímos um modelo de ML para classificação (com learning to rank) dos clientes mais propensos a contratar o novo seguro. Como resultado geramos uma lista ordenada com os melhores clientes no topo, tal ordenação garantiu acessar 47% dos interessados contatando apenas 15% da base. Ou ainda, acessar 81% dos interessados contatando 32% da base.

    Identificação de Clientes de Alto Valor

    Este é um projeto de Ciências de Dados que utliza técnicas de agrupamento (clusterização) de clientes de um E-Commerce. O objetivo do projeto é identificar um grupo de clientes que seja mais importantes para a empresa para que seja fornecido um programa de fidelidade. Foram utilizados modelos de aprendizagem não supervisionado para dividir os clientes em 10 grupos, sendo de um deles claramente se destacou em relação aos outros e o nomeamos com Grupo Insiders. O modelo foi colocado em produção via AWS, utilizando ferramentas como S3, EC2 e RDS (Postgres). Também foi gerado um Dashboard com os principais insights desse grupo de clientes.

    As ferramentas utilizadas foram:

    • Python, Pandas, Numpy, Seaborn.
    • Jupyter Notebook.
    • Agrupamento (Clusterização): K-Means, GMM, Clusterização Hierárquica, DBSCAN
    • Redução de dimensionalidade: PCS, t-SNE, UMAP
    • Embedding de árvore
    • AWS Amazon: S3, EC2, RDS.
    • Banco de Dados: SQLite, Postgres.
    • Cronjob e Papermill.
    • Looker Studio
